Сайт "High pass".

Реализованы:
- CSS Grid.
- Верстка по pixel perfect.
- Адаптив сайта.
- Gulp-сборка
- Scss.
- Бургер-меню.
- Окно поиска на странице.
- Переход по странице из Header.
- Именование классов по методологии БЭМ.
- Все состояния интерактивных элементов: hover, focus, active
- API map.
- Валидация формы.


Для работы необходимо скачать репозиторий.
Выполнить команду git init.

Для сборки проекта выполнить в терминате команду gulp.
